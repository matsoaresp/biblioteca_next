# 📚 Sistema de Biblioteca (Full Stack)

Este é um projeto full stack desenvolvido com **Next.js** no front-end e **NestJS** no back-end, com autenticação baseada em **JWT**.  
A aplicação simula um sistema de biblioteca, com foco em exibição de livros e gerenciamento de usuários.

## 🚀 Deploy

A aplicação está em produção utilizando o **Vercel**.


## 🧠 Tecnologias utilizadas

### Front-end
- Next.js  
- React  
- Tailwind CSS  

### Back-end
- NestJS  
- Node.js  
- JWT (JSON Web Token)  


## 🔐 Autenticação

O sistema utiliza autenticação baseada em **JWT**, garantindo segurança no processo de login e controle de sessão do usuário.

## 📚 Funcionalidades

- 📖 Listagem de livros (dados mockados)  
- 🎠 Carrossel de livros na página inicial  
- 🔎 Visualização de sinopse ao clicar em um livro  
- 🔐 Sistema de login  
- 👤 Exibição dos dados do usuário no dropdown após login  

## ⚠️ Limitações atuais

- Os livros exibidos são apenas ilustrativos (mockados no código)  
- Não é possível adicionar, editar ou remover livros via interface  
- Não há persistência de dados em banco para os livros
